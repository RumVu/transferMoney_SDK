Welcome

Documentation for teams shipping currency conversion features

transferMoney SDK ships with a product-style documentation portal for onboarding, release selection, and operational guidance, while still exposing full DocC API reference for engineers who need symbol-level details.

Get Started Open API Reference
Features

What the SDK supports

Supported currencies

VND, USD, and AUD with direct and multi-step conversion paths.

Deterministic runtime

No network dependency. All conversion logic runs locally inside the app.

Flexible rates

Use configured defaults, runtime updates, or one-off custom rates per call.

Release-aware docs

Developers can choose a stable version instead of blindly following the latest tag.

Supported conversion matrix VND ↔ USD, USD ↔ AUD, and VND ↔ AUD through USD.

Examples

Quick sample

import transferMoney_SDK

let converter = TransferMoney_core()
let result = try converter.convert(
  amount: 1_000_000,
  from: .VND,
  to: .USD,
  choose: .standard
)

print(result.targetAmount)
print(result.FormattedTargetAmount)
FAQ

Common questions

Do I have to use the newest release?

No. Start with 0.0.9 for new work, or pin 0.0.8 if your production app already validated that version.

Where is the API reference?

Open DocC API Reference for classes, methods, and symbol-level details.

Does the SDK need internet access?

No. Conversion logic runs locally in the client app.
