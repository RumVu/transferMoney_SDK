Convert VND, USD, and AUD inside your app with a lightweight Swift SDK that ships with DocC-ready guides, release notes, and version selection help.

## Overview

`transferMoney_SDK` is built for teams that need deterministic currency conversion logic without network calls. The package supports direct VND/USD and USD/AUD conversions plus automatic multi-step VND/AUD flows through USD.

Use this documentation set as the main entry point for onboarding, evaluating versions, and integrating the SDK into production projects.
