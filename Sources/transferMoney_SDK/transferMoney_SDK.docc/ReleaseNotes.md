# Release Notes

Use this page as the high-level changelog inside DocC. For the full historical file, see the repository `CHANGELOG.md`.

## 0.0.9

_April 17, 2026_

- Added a full DocC catalog with curated landing pages and sidebar navigation.
- Added Swift Package Index documentation metadata through `.spi.yml`.
- Added version selection guidance so integrators can choose a stable release intentionally instead of defaulting to the newest tag.
- Fixed DocC source references that produced warnings during documentation generation.

## 0.0.8

_April 17, 2026_

- Fixed static-hosting deployment flow by separating generated output from authored docs content.
- Stopped mixing generated DocC files into the `docs/` source folder.

## 0.0.7

_April 17, 2026_

- Added release and version-selection guidance for package consumers.

## 0.0.6

_April 17, 2026_

- Added the first large set of authored guides and examples for the SDK.
